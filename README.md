# Portfolio

[![test](https://github.com/DevelopersWork/portfolio/actions/workflows/test.yaml/badge.svg?branch=testing)](https://github.com/DevelopersWork/portfolio/actions/workflows/test.yaml)
[![build](https://github.com/DevelopersWork/portfolio/actions/workflows/build.yaml/badge.svg?branch=main)](https://github.com/DevelopersWork/portfolio/actions/workflows/build.yaml)

A fully responsive personal portfolio website built with React.js, designed to showcase skills, experience, and projects in a professional and interactive way.

## Features

-   **Responsive Design**: Fully responsive layout that adapts to all screen sizes.
-   **Interactive UI**: Utilizing Material UI and Styled Components for a modern look.
-   **Contact Form**: Integrated with EmailJS for direct communication.
-   **Smooth Navigation**: Single-page application with smooth scrolling navigation.
-   **Dynamic Animations**: Engaging hero section and background animations.

## Tech Stack

-   **Frontend**: [React](https://reactjs.org/)
-   **Routing**: [React Router](https://reactrouter.com/)
-   **UI Library**: [Material UI](https://mui.com/)
-   **Styling**: [Styled Components](https://styled-components.com/), [Emotion](https://emotion.sh/)
-   **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
-   **Animations**: [Typewriter Effect](https://github.com/ Takuyaweb/react-typewriter-effect)
-   **Email Service**: [EmailJS](https://www.emailjs.com/)

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

-   Node.js (v14 or later recommended)
-   npm

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/DevelopersWork/portfolio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd portfolio
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Usage

**Development Server:**
Runs the app in the development mode.
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

**Build for Production:**
Builds the app for production to the `build` folder.
```bash
npm run build
```

**Run Tests:**
Launches the test runner in the interactive watch mode.
```bash
npm test
```

## Deployment

This project is configured for deployment to **GitHub Pages**.

To deploy the latest build:
```bash
npm run deploy
```
